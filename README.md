# Оно тебе надо

Проект демонстрации современных подходов к frontend-разработке

**Одностраничное веб-приложение** — демонстрация современных подходов к верстке с использованием семантического HTML5, CSS Grid и Flexbox, а также небольшой оптимизации производительности фронтенда.

## Описание проекта

Данный проект представляет собой пример реализации landing page с применением современных стандартов веб-разработки. Основное внимание уделяется:

- Архитектуре CSS с модульной организацией стилей
- Семантической HTML-разметке для улучшения доступности
- Оптимизации загрузки ресурсов и производительности
- Адаптивной верстке без использования JavaScript
